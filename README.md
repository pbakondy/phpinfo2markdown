# Convert phpinfo() to markdown

## Usage

```
$ npm i -g phpinfo2markdown
```

First you have to save your phpinfo page to html file.

```
// conversion
$ phpinfo2markdown myPhpInfo.html myPhpInfo.md

// help page
$ phpinfo2markdown
```
